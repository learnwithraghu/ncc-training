# Demo Infra Requirement

## Infra Needed
- Linux shell environment
- Standard CLI tools: ls, cat, grep, find, chmod, sudo
- Learner account with permission to create files and folders

## Quick Validation
```bash
uname -a
whoami
command -v ls cat grep find chmod
mkdir -p /tmp/ncc-linux-check && touch /tmp/ncc-linux-check/ok && ls -la /tmp/ncc-linux-check
```
