# Exercise: Basic Navigation

## Goal
Practice the commands you need to move around the Linux file system.

## Tasks
1. Open a terminal.
2. Print your current directory with `pwd`.
3. List the files in the current directory with `ls`.
4. Show hidden files with `ls -a`.
5. Change to your home directory with `cd ~`.
6. Move to the root directory with `cd /`.
7. Return to the directory you were in before with `cd -`.

## Check Yourself
- You should be able to explain what `~`, `/`, and `-` mean in the context of `cd`.
