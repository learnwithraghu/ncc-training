# Topic 1: Basic Navigation

**Time:** 20 minutes

## Goal
Move around the Linux file system with confidence.

## Commands to Use
```bash
pwd
ls
ls -a
cd ~
cd /
cd -
clear
```

## Guided Steps
1. Run `pwd` and say where you are.
2. Run `ls` and `ls -a` and compare the output.
3. Change to your home directory with `cd ~`.
4. Go to the root directory with `cd /`.
5. Return to the previous directory with `cd -`.

## Checkpoint
Explain the difference between `~`, `/`, and `-`.
