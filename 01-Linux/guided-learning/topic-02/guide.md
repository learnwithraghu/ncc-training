# Topic 2: Directory Structure

**Time:** 20 minutes

## Goal
Create nested folders and understand absolute versus relative paths.

## Commands to Use
```bash
mkdir sandbox
mkdir -p sandbox/notes/today
cd sandbox/notes/today
pwd
cd ~/sandbox
ls
```

## Guided Steps
1. Create a `sandbox` directory.
2. Build a nested path with `mkdir -p`.
3. Move into the deepest directory using a relative path.
4. Jump back to `~/sandbox` with an absolute path.
5. Use `pwd` to confirm where you are.

## Checkpoint
Describe when you would use a relative path and when you would use an absolute path.
