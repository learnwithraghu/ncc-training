# Topic 3: Help and Man Pages

**Time:** 20 minutes

## Goal
Use built-in help to learn a command without leaving the terminal.

## Commands to Use
```bash
man ls
/hidden
n
q
mkdir --help
man chmod
```

## Guided Steps
1. Open `man ls` and find the `-a` option.
2. Search the page for `hidden` with `/hidden`.
3. Press `n` to jump to the next match.
4. Exit with `q`.
5. Run `mkdir --help` and compare it with `man mkdir`.

## Checkpoint
Tell me the difference between `man` and `--help`.
