# Topic 4: Hidden Files and Globbing

**Time:** 20 minutes

## Goal
Work with hidden files and file patterns.

## Commands to Use
```bash
touch .env alpha.txt beta.txt notes.md
ls
ls -a
ls *.txt
ls *.md
```

## Guided Steps
1. Create a hidden file called `.env`.
2. Create a few normal files with different extensions.
3. Use `ls` and `ls -a` to compare visible versus hidden files.
4. Use wildcards like `*.txt` and `*.md`.
5. Explain which files each pattern matches.

## Checkpoint
Why does a dot file not show up in a normal `ls`?
