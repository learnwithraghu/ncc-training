# Topic 5: File Management

**Time:** 20 minutes

## Goal
Create, copy, move, and remove files safely.

## Commands to Use
```bash
mkdir practice
cd practice
touch notes.txt todo.txt
cp notes.txt notes-backup.txt
mv todo.txt tasks.txt
rm notes-backup.txt
cat notes.txt
```

## Guided Steps
1. Make a practice folder.
2. Create two empty files.
3. Copy one file to a backup name.
4. Rename the other file.
5. Remove the backup file.

## Checkpoint
Which command renames a file, and which command deletes it?
