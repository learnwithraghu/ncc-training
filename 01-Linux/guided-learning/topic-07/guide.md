# Topic 7: Viewing Files and Text Output

**Time:** 20 minutes

## Goal
Inspect file contents quickly from the terminal.

## Commands to Use
```bash
seq 1 20 > sample.txt
cat sample.txt
head sample.txt
tail sample.txt
less sample.txt
```

## Guided Steps
1. Create a sample text file with multiple lines.
2. Show the whole file with `cat`.
3. Read just the start with `head`.
4. Read just the end with `tail`.
5. Open the file in `less` and quit with `q`.

## Checkpoint
When would you use `less` instead of `cat`?
