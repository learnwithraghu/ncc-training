# Topic 8: Editing with Nano and Vim

**Time:** 20 minutes

## Goal
Edit a text file using two common terminal editors.

## Commands to Use
```bash
touch notes.txt
nano notes.txt
vim notes.txt
cat notes.txt
```

## Guided Steps
1. Create a blank file.
2. Open it in `nano` and type one line.
3. Save and exit `nano`.
4. Open it in `vim`, press `i`, add another line, then press `Esc` and `:wq`.
5. Confirm the final content with `cat`.

## Checkpoint
Which keys do you press in `vim` to save and quit?
