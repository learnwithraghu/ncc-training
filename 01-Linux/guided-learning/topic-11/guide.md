# Topic 11: Disk Usage and System Info

**Time:** 20 minutes

## Goal
Check disk space, memory, and system details.

## Commands to Use
```bash
df -h
du -sh .
free -h
uname -a
whoami
```

## Guided Steps
1. Check available disk space with `df -h`.
2. Check the size of your current folder with `du -sh .`.
3. Review memory usage with `free -h`.
4. Print system details with `uname -a`.
5. Confirm which user you are with `whoami`.

## Checkpoint
Which command would you use first if a machine is running out of disk space?
