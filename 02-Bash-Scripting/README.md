# Day 1, Part 2: Bash Scripting

This module is the second part of **Day 1** of the NCC DevOps Bootcamp. It teaches you how to automate Linux tasks using Bash scripts.

## What You Will Learn

By the end of this module, you will be able to:

- Write and execute Bash scripts
- Use variables, user input, and conditions
- Create loops, functions, and command substitution
- Build real automation scripts for backup and reporting

## Time Estimate

Approximately **3 hours** total, split into 10 guided topics at about 20 minutes each.

## Prerequisites

- Completion of [01-Linux](../01-Linux/README.md) guides
- Working directory: `~/ncc-labs/day1/`

The `backup.sh` script you build here is part of your Day 1 toolkit. On Day 2, you will commit this entire toolkit to a GitHub repository.
