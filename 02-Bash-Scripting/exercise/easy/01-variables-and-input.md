# Exercise: Variables and Input

## Goal
Practice Bash variables, user input, and simple conditionals.

## Tasks
1. Create a script named `greet.sh`.
2. Ask the user for their name.
3. Store the response in a variable.
4. Print a greeting that includes the name.
5. Add a simple check that prints a different message if the name is empty.

## Check Yourself
- You should be able to use `read`, variable expansion, and `if`.
