# Exercise: Backup Automation

## Goal
Build a small backup script for text and log files.

## Tasks
1. Create a script named `backup.sh`.
2. Accept a source directory as an argument.
3. Create a timestamped backup directory.
4. Copy `.txt` and `.log` files into the backup directory.
5. Print a message that shows where the backup was created.

## Check Yourself
- You should be able to use command-line arguments, `mkdir -p`, command substitution, and file globbing.
