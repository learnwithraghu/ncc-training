# Exercise: Loops and Functions

## Goal
Practice looping through values and wrapping repeated logic in functions.

## Tasks
1. Create a script named `repeat.sh`.
2. Write a function that prints a short status message.
3. Use a `for` loop to call the function for three sample items.
4. Print the current item inside the function.

## Check Yourself
- You should know how to define a function and how a `for` loop feeds values into it.
