# Lab 01 Steps

## Step 1: Inspect the script
Run:
```bash
cd ~/ncc-labs/day1/bash-lab-01
ls -l
cat hello.sh
```
Point out the shebang at the top of the file.

## Step 2: Try to run it
Run:
```bash
./hello.sh
```
Explain why the command fails before the permission fix.

## Step 3: Make it executable
Run:
```bash
chmod +x hello.sh
ls -l hello.sh
```

## Step 4: Run the script again
Run:
```bash
./hello.sh
```
Confirm the output appears in the terminal.

## Checkpoint
The learner should be able to explain what a shebang is and why execute permission matters.
