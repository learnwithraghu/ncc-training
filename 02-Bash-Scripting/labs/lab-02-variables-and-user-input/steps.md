# Lab 02 Steps

## Step 1: Open the script
Run:
```bash
cd ~/ncc-labs/day1/bash-lab-02
cat greet.sh
```
Review the variables and `read` commands.

## Step 2: Run the script
Run:
```bash
./greet.sh
```
Enter a name and a project code when prompted.

## Step 3: Change the input values
Run the script again with different values.
Explain that the output changes based on the variables.

## Step 4: Add a simple echo test
Run:
```bash
name="Asha"
code="NCC101"
echo "Hello $name, welcome to project $code"
```
Show how variables can also be assigned directly in the shell.

## Checkpoint
The learner should be able to use `read` and variables to capture and display information.
