# Demo Infra Requirement

## Infra Needed
- Python 3.10+
- pip for package installation
- Virtual environment support (venv)

## Quick Validation
```bash
python3 --version
python3 -m pip --version
python3 -m venv /tmp/ncc-py-venv && source /tmp/ncc-py-venv/bin/activate && python --version && deactivate
```
