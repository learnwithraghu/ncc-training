# Lab 01: Python Setup and First Script

## Why this lab
Learners need to confirm Python is installed, create a script, and run it successfully.

## Scenario details
The learner starts with a clean Python workspace and one starter script that prints a message. The goal is to confirm the environment, inspect the script, and execute it from the terminal.

## Lab setup
- Python 3 is available on the machine
- A starter script exists in the lab folder
- The learner has a safe place to run `python3`

## Suggested engineer setup
```bash
mkdir -p ~/ncc-labs/day1/python-lab-01
cat > ~/ncc-labs/day1/python-lab-01/hello.py <<'EOF'
print("hello from python")
EOF
```

## Success criteria
- Verify the Python version
- Inspect the script
- Run the script with `python3`
