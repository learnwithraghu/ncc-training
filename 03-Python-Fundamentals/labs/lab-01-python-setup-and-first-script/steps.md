# Lab 01 Steps

## Step 1: Check Python
Run:
```bash
python3 --version
```
Confirm that Python 3 is installed.

## Step 2: Inspect the starter script
Run:
```bash
cd ~/ncc-labs/day1/python-lab-01
cat hello.py
```
Point out the single print statement.

## Step 3: Run the script
Run:
```bash
python3 hello.py
```
Explain that Python scripts are executed with the interpreter.

## Step 4: Make a small edit
Run:
```bash
echo 'print("lab complete")' > hello.py
python3 hello.py
```
Show how changing the file changes the output.

## Checkpoint
The learner should know how to verify Python, inspect a script, and run it with `python3`.
