# Topic 1: Container Mindset and Quick Test

**Time:** 20 minutes

## Goal
Understand why containers matter and verify that Docker is working.

## Commands to Use
```bash
docker --version
docker info
docker run hello-world
```

## Guided Steps
1. Check the installed Docker version.
2. Review `docker info` to confirm the daemon is running.
3. Run `hello-world`.
4. Explain what happened when Docker pulled and started the container.
5. Compare the container model to a traditional virtual machine.

## Checkpoint
What is the main advantage of containers over full virtual machines?
