# Topic 12: Docker Compose Basics

**Time:** 20 minutes

## Goal
Learn how Compose defines and runs services from a file.

## Commands to Use
```bash
cd /workspaces/ncc-training/05-Docker/application
vi docker-compose.yml
docker compose config
docker compose up -d
docker compose down
```

## Guided Steps
1. Open a Compose file in `vi`.
2. Define a service for the sample app.
3. Validate the Compose file with `docker compose config`.
4. Start and stop the app with Compose.
5. Explain how Compose fits local development.

## Checkpoint
What problem does Docker Compose solve for a developer or instructor?
