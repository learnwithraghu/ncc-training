# Guided Learning - Kubernetes

This module uses one self-contained lesson per topic.

## Structure

- topic-01 through topic-20 hold the learning topics
- each topic has one guide.md file
- topics that need manifests or scripts include a local assets/ folder
- each topic is designed for about 20 minutes

## Recommended Flow

1. Open the topic guide.
2. Read the explanation and commands.
3. Run the commands as you go.
4. Answer the checkpoint prompt.
5. Move to the next topic.
